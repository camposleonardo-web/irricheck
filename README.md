# IrriCheck — Agricef

App de avaliação de assertividade e espaçamento do irrigador de mudas de eucalipto.

## Como publicar (GitHub Pages — gratuito)

1. Crie uma conta em https://github.com (se não tiver)
2. Clique em **New repository** → nome: `irricheck` → Public → Create
3. Clique em **uploading an existing file**
4. Arraste todos os arquivos desta pasta: `index.html`, `manifest.json`, `sw.js`, `icon-192.png`, `icon-512.png`
5. Clique em **Commit changes**
6. Vá em **Settings** → **Pages** → Source: **main** → **Save**
7. Aguarde ~1 minuto. O app estará em: `https://SEU-USUARIO.github.io/irricheck`
8. Abra esse link no Chrome do celular → aparecerá a opção de instalar!

## Uso offline
Após instalado, funciona 100% sem internet.
